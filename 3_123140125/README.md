## 📱 Fitur Aplikasi

Aplikasi ini memiliki beberapa fitur utama:

- Foto profil berbentuk **circular**
- Nama dan **NIM**
- Deskripsi singkat / bio
- Informasi kontak:
  - Email
  - Nomor telepon
  - Lokasi
- Tombol **Contact / Detail Profile**

---

## 🧩 Komponen UI yang Digunakan

Dalam aplikasi ini digunakan beberapa komponen UI dari **Jetpack Compose**, yaitu:

- **Column** → menyusun komponen secara vertikal  
- **Row** → menyusun komponen secara horizontal  
- **Box** → container untuk menempatkan elemen UI  
- **Card** → menampilkan informasi profil dalam bentuk kartu  
- **Text** → menampilkan teks  
- **Button** → tombol aksi  
- **Image** → menampilkan foto profil  
- **Icon** → ikon untuk informasi kontak  

---

## 🔁 Reusable Composable Functions

Project ini menggunakan beberapa **Composable Function yang reusable**, yaitu:

### 1. `ProfileHeader()`
Menampilkan bagian header aplikasi yang berisi background berwarna hijau dan foto profil.

### 2. `ProfileCard()`
Menampilkan informasi utama pengguna seperti nama, NIM, bio, dan tombol aksi.

### 3. `InfoItem()`
Menampilkan setiap item informasi kontak seperti email, nomor telepon, dan lokasi.
