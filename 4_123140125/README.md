## 📌 Deskripsi Tugas

Project ini merupakan pengembangan dari aplikasi **Profile App** pada minggu sebelumnya dengan penambahan fitur:

1. **Implementasi MVVM Pattern**
   - Menggunakan `ViewModel`
   - Menggunakan `StateFlow`
   - Menggunakan data class `ProfileUiState`

2. **Fitur Edit Profile**
   - Form untuk edit nama dan bio
   - Menggunakan state hoisting pada `TextField`
   - Tombol **Save** untuk update data ke ViewModel

3. **Fitur Dark Mode Toggle**
   - Switch untuk dark/light mode
   - State disimpan di ViewModel
