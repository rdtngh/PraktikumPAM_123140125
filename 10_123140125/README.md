# Notes App - DI & Testing

## Deskripsi
Project Notes App berbasis Android menggunakan:
- Koin Dependency Injection
- MVVM Architecture
- Unit Testing
- Flow Testing
- UI Testing Jetpack Compose

---

# Fitur Tugas yang Diimplementasikan

## 1. Dependency Injection (Koin)
Menggunakan 2 module:
- Data Module
- ViewModel Module

## 2. Repository Testing
✅ Minimal 5 test cases

Contoh:
- Add Note
- Delete Note
- Update Note
- Get Notes
- Empty Validation

## 3. ViewModel Testing
✅ Minimal 4 test cases menggunakan MockK

Contoh:
- Load Notes
- Add Note
- Delete Note
- Error State

## 4. Flow Testing
✅ Minimal 2 test cases menggunakan Turbine

Contoh:
- Flow emit data
- Flow emit empty state

## 5. UI Testing
✅ Minimal 3 test cases menggunakan Compose Test

Contoh:
- Display Notes
- Add Button Click
- Navigation Test

## 6. Code Coverage
✅ Minimum 60% business logic coverage
